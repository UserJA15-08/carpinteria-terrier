Cómo subirlo con cPanel:
1. Entra al cPanel del hosting.
2. Abre File Manager.
3. Entra a public_html o a la carpeta raíz del dominio.
4. Sube el contenido completo de esta carpeta.
5. Si subes el .zip, extrae su contenido dentro de public_html.
6. Verifica que index.html quede directamente en la raíz visible del sitio.

Cómo subirlo con GitHub Pages:
1. Crea un repositorio nuevo.
2. Sube el contenido completo de esta carpeta.
3. Activa Pages desde Settings > Pages.
4. Publica desde la rama principal y carpeta root.
